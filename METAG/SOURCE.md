# METAG — original pipeline by Julia (Birch) Klomparens

Source: https://github.com/bklomparens/METAG

Birch's original per-step metagenomics pipeline (trimmomatic -> fastqc ->
megahit -> mapping/coverm -> metabat2 -> checkm -> prokka -> cgview).
This is the predecessor to the automated Snakemake pipeline in this repo.
Preserved here for provenance and reference.
